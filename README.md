<h1 style="font-size: 36px;">EmotiTrack</h1>

El primer producto que desarrollamos es EmotiTrack, una aplicación multiplataforma (web y móvil) que surge como una propuesta innovadora en el campo del bienestar digital. EmotiTrack se enfoca en una problemática cada vez más visible en la sociedad contemporánea: la falta de herramientas prácticas, accesibles y motivadoras para gestionar las emociones en la vida cotidiana.
